# InfoTrack - Frontend

Interfaz web del sistema InfoTrack, enfocada en la visualización de información académica para estudiantes de informática.

## 📎 Descripción

Este proyecto corresponde al frontend de InfoTrack. Actualmente implementa una landing page donde los usuarios pueden explorar información académica sin necesidad de autenticación.

La aplicación está pensada como una SPA (Single Page Application) y en futuras versiones se conectará a una API GraphQL.

## 📎 Funcionalidades actuales

- Visualización de carreras disponibles
- Listado de materias
- Filtro de materias por carrera
- Visualización del cuatrimestre activo
- Recomendaciones de materias
- Interfaz responsive orientada a distintos dispositivos

## 📎 Tecnologías utilizadas

- React
- Vite
- HTML5
- CSS3

## 📎 Estado del proyecto
